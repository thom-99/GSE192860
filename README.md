### Differential Gene Expression and Gene Function Enrichment Analyses for the study titled 'Antipsychotic Drug Trifluoperazine as a Potential Therapeutic Agent against Nasopharyngeal Carcinoma' 

the data is available on GEO already in the form of Transcripts per Million 

the analyses aims to be reproducible: in the jupyter notebook you'll find some links that clarify some concepts which are useful to understand

If you have any idea in how to improve the repo please feel free to share, I am still in the process of learning omics data analysis and I'd be happy to take suggestions.
